##  Installation

  

Step-by-step instructions to set up the project locally:
1. Clone the repository:

        git clone https://github.com/Roshan-Metrix/hamburger.git

  
2. Navigate to the project directory:

       cd hamburger

4. Run App

       Open in any browser
